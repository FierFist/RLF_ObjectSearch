# 物品预设（Preset）配置说明

## 配置结构

```json
[
    {
        "presetID": "ExamplePreset1",
        "listOfItemsID": {
            "242": 1,
            "245": 1,
            "246": 1,
            "247": 1,
            "248": 1
        }
    }
]
```

---

## 参数说明

| 参数 | 类型 | 示例值 | 说明 |
|------|------|--------|------|
| **presetID** | `String` | `ExamplePreset1` | 预设的唯一标识符 |
| **listOfItemsID** | `Object {String: Integer/Float}` | 见下方 | 包含物品 ID 及其掉落权重的列表 |

---

## 物品列表（示例）

| Item ID | 权重 | 掉落概率 | 说明 |
|---------|------|----------|------|
| `242` | `1` | 20% (1/5) | 物品 ID 242 |
| `245` | `1` | 20% (1/5) | 物品 ID 245 |
| `246` | `1` | 20% (1/5) | 物品 ID 246 |
| `247` | `1` | 20% (1/5) | 物品 ID 247 |
| `248` | `1` | 20% (1/5) | 物品 ID 248 |

---

## 权重工作原理

**计算公式：**

```
物品概率 = （该物品权重 ÷ 所有权重之和）× 100%
```

**示例：**

- 总权重 = `1 + 1 + 1 + 1 + 1 = 5`
- 每个物品概率 = `1 / 5 = 0.2 = 20%`

---

## 不同权重示例

### 示例 1：不同概率

```json
{
    "presetID": "WeightedPreset",
    "listOfItemsID": {
        "242": 5,    // 50% (5/10)
        "245": 3,    // 30% (3/10)
        "246": 2     // 20% (2/10)
    }
}
```

| Item ID | 权重 | 概率 |
|---------|------|--------|
| `242` | `5` | 50% |
| `245` | `3` | 30% |
| `246` | `2` | 20% |

---

### 示例 2：稀有度不同的物品

```json
{
    "presetID": "RareItemsPreset",
    "listOfItemsID": {
        "100": 10,   // 76.92% (10/13) - 常见
        "101": 2,    // 15.38% (2/13) - 不常见
        "102": 1     // 7.69% (1/13) - 稀有
    }
}
```

| Item ID | 权重 | 概率 | 稀有度 |
|---------|------|--------|----------|
| `100` | `10` | 76.92% | 常见 |
| `101` | `2` | 15.38% | 不常见 |
| `102` | `1` | 7.69% | 稀有 |

## 预设的使用方式

预设可在“可搜刮物体配置”中使用：

```json
{
    "RLF_Lootable_Object": {
        "itemListPreset": ["ExamplePreset1"],
        "chanceToSuccess": 0.5
    }
}
```

---

## 多重预设示例

你可以为一个对象指定多个预设：

```json
{
    "presetID": "FoodPreset",
    "listOfItemsID": {
        "1": 1,
        "2": 1
    }
},
{
    "presetID": "ToolsPreset",
    "listOfItemsID": {
        "10": 1,
        "11": 1
    }
}
```

使用方式：

```json
"itemListPreset": ["FoodPreset", "ToolsPreset"]
```

---

## 注意事项

- 权重可以使用整数或小数  
- 权重越大，掉落概率越高  
- 权重总和不需要为 100  
- 预设方便多个对象共享同一组合的掉落物  
- 同一个预设可以被多个对象重复引用
