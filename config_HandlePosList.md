# 建筑内物体生成配置
## 用途说明

该配置用于在无法通过标准 Proxy 方式放置物体的情况下，在建筑内部生成家具和其他对象。

## 配置结构示例

```json
{
    "Land_Village_Pub": [
        "RLF_Lootable_PlasticCanister|5.364221 -10.644337 5.471502|0.000000 90.000000 -0.000000"
    ],
    "Land_HouseBlock_2F1": [
        "RLF_Lootable_PlasticCanister|5.364221 -10.644337 5.471502|0.000000 90.000000 -0.000000"
    ]
}

## 格式说明

| 元素 | 描述 | 示例 |
|------|------|------|
| **对象键** | 建筑类名，在该建筑内生成物体 | `Land_Village_Pub` |
| **对象数组** | 要在该建筑中生成的物体列表 | `[ "object1", "object2" ]` |

---

## 物体字符串格式

物体字符串由 `|` 分成三部分：
ClassName|X Y Z|Pitch Yaw Roll

---

| 组件 | 类型 | 描述 | 示例 |
|------|------|----------|----------|
| **ClassName** | String | 要生成物体的类名 | `RLF_Lootable_PlasticCanister` |
| **坐标 (X Y Z)** | Float Float Float | 物体相对建筑中心的位置（米） | `5.364221 -10.644337 5.471502` |
| **方向 (Pitch Yaw Roll)** | Float Float Float | 物体的旋转角度（度） | `0.000000 90.000000 -0.000000` |

---

## 坐标说明

| 轴 | 描述 |
|----|----------|
| **X** | 左右方向的偏移 |
| **Y** | 高度方向的偏移 |
| **Z** | 前后方向的偏移 |

---

## 旋转说明

| 角度 | 描述 | 范围 |
|--------|----------|----------|
| **Pitch** | 前后倾斜 | -180° 到 180° |
| **Yaw** | 左右旋转 | 0° 到 360° |
| **Roll** | 左右侧倾 | -180° 到 180° |

---

## 使用示例

```json
{
    "Land_Village_Pub": [
        "RLF_Lootable_PlasticCanister|5.364221 -10.644337 5.471502|0.000000 90.000000 -0.000000",
        "Land_Chair_2|3.5 -10.5 4.0|0.000000 45.000000 0.000000"
    ],
    "Land_HouseBlock_2F1": [
        "RLF_Lootable_PlasticCanister|5.364221 -10.644337 5.471502|0.000000 90.000000 -0.000000",
        "Land_Table_1|2.0 -10.0 3.5|0.000000 0.000000 0.000000"
    ]
}
