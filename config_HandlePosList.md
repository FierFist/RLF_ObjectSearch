# 建筑内物体生成配置
## 用途说明

该配置用于在无法通过标准 Proxy 方式放置物体的情况下，在建筑内部生成家具和其他对象。

## 配置结构示例

```json
{
    "Land_Village_Pub": [
        "RLF_Lootable_PlasticCanister|5.364221 -10.644337 5.471502|0.000000 90.000000 -0.000000"
    ],
    "Land_HouseBlock_2F1": [
        "RLF_Lootable_PlasticCanister|5.364221 -10.644337 5.471502|0.000000 90.000000 -0.000000"
    ]
}

## 格式说明

| 元素       | 描述             | 示例                         |
| -------- | -------------- | -------------------------- |
| **对象键**  | 建筑类名，在该建筑内生成物体 | `Land_Village_Pub`         |
| **对象数组** | 要在该建筑中生成的物体列表  | `[ "object1", "object2" ]` |
