# 配置：建筑物内物体刷生成（HandlePosList）

本配置用于在建筑物内部生成家具、战利品或其他对象，适用于无法通过 DayZ 代理物体（Proxy）系统摆放的情况。

---

## 配置结构

```json
{
    "Land_Village_Pub": [
        "RLF_Lootable_PlasticCanister|5.364221 -10.644337 5.471502|0.000000 90.000000 -0.000000"
    ],
    "Land_HouseBlock_2F1": [
        "RLF_Lootable_PlasticCanister|5.364221 -10.644337 5.471502|0.000000 90.000000 -0.000000"
    ]
}
```

## 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 对象键（建筑类名） | 要在哪个建筑内生成物体 | `Land_Village_Pub` |
| 对象数组 | 需要在此建筑内生成的所有物体条目 | `[ "Object1", "Object2" ]` |

---

## 对象条目格式

每一条对象字符串使用 `|` 分为 3 个部分：

```
ClassName|X Y Z|Pitch Yaw Roll
```

| 部分 | 类型 | 说明 | 示例 |
|------|------|------|------|
| ClassName | String | 生成的物体类名 | `RLF_Lootable_PlasticCanister` |
| 坐标 (X Y Z) | Float Float Float | 相对于房屋中心的偏移坐标（米） | `5.364221 -10.644337 5.471502` |
| 旋转角度 (Pitch Yaw Roll) | Float Float Float | 欧拉角旋转（度） | `0.000000 90.000000 -0.000000` |

---

## 坐标说明

| 轴 | 意义 |
|----|------|
| X | 左右偏移 |
| Y | 上下高度偏移 |
| Z | 前后偏移 |

---

## 旋转角度说明

| 角度 | 意义 | 范围 |
|------|------|------|
| Pitch | 前后倾斜 | -180° ~ 180° |
| Yaw | 左右旋转 | 0° ~ 360° |
| Roll | 左右横滚 | -180° ~ 180° |

---

## 示例配置

```json
{
    "Land_Village_Pub": [
        "RLF_Lootable_PlasticCanister|5.364221 -10.644337 5.471502|0.000000 90.000000 -0.000000",

    ],
    "Land_HouseBlock_2F1": [
        "RLF_Lootable_PlasticCanister|5.364221 -10.644337 5.471502|0.000000 90.000000 -0.000000",
        "Land_Table_1|2.0 -10.0 3.5|0.000000 0.000000 0.000000"
    ]
}
```
